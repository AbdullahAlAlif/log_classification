About Log Classification System

The Log Classification System is a powerful tool designed to help you classify and analyze log files effortlessly. Whether you're a system administrator, developer, or analyst, this tool provides insights into your logs with ease.

Upload your log files, classify them, and view detailed results. Registered users can also save their uploads and access them anytime.

The system is built using Python and Flask, ensuring a smooth and efficient user experience.

  • It uses Regular Expressions (Regex) for fast classification of logs that follow patterns.
  • It uses Logistic Regression (ML algorithm) for classification of logs that don't follow patterns. We used Sentence Transformer to create embeddings of logs and train the model on them.
  • It uses the Ollama open-source LLM model for classification of logs that don't have enough data points to train on.

Thus, it ensures a balance of accuracy and speed.

The system was trained on a dataset that can classify logs into 5 categories (Security Alert, Error, HTTP Status, Workflow, Deprecations). The rest will produce "Unknown." However, it can be developed for other datasets with some fine-tuning, making it a versatile solution for different log formats and structures.
